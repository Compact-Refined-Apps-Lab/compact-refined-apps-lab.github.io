How can we help?

FoleyCue is a paid-once, offline spotting-sheet, pull-list, and take-logging utility for iPhone and iPad. It requires no account, subscription, network connection, or external hardware.

Start

Create a project and add cues

Create a project for a production, reel, or scene, choose the intended timecode rate, then add cues with manually entered start and end timecodes. Keep actions, categories, characters, props, surfaces, performers, status, notes, and pickup flags together.

Timecode

Check the rate before the session

FoleyCue supports common 23.976–60 fps timecode choices, including supported non-drop and drop-frame modes. It does not read picture, EDL/XML, LTC, or MTC. Always confirm the project rate and every timecode against the production source.

Prep

Use the pull list and handoff tools

Props and surfaces entered on cues feed the project pull list. Search and filters help narrow the cue sheet. Export CSV, text, or a FoleyCue JSON backup through Apple's share sheet only when you choose a destination.

Run

Log takes manually

Live Run keeps the current and next cue visible. Log repeated takes, rate the latest take, flag a pickup, and advance with explicit taps. FoleyCue does not record audio or video, synchronize to equipment, trigger hardware, or control a show.

Import

Validate imported files

CSV cue-sheet import and FoleyCue JSON restore read only files you select. Keep an untouched backup and review every imported cue, frame rate, timecode, note, and export before relying on it in a session or delivery.

Contact

Still need help?

Email compactrefinedapps@outlook.com and include your device model, iOS or iPadOS version, FoleyCue version, and a short description. Do not attach confidential production material, passwords, payment information, or other sensitive information.
