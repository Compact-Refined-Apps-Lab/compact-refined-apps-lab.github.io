How can we help?

CelCue is a paid-once, offline X-sheet and animation-timing planning workspace for iPhone and iPad. It requires no account, subscription, network connection, production media, or external hardware.

Start

Create a scene plan

Create a project and scene, choose 12, 15, 24, 25, 30, 48, or 60 fps, set the scene length, and add character, effects, camera, background, or custom exposure columns. A fictional example project is included for exploration.

Expose

Plan drawings and holds

Add custom drawing blocks or generate runs on 1s, 2s, or 3s. The vertical X-sheet shows drawing changes and held-drawing notation frame by frame. Review every drawing label, range, and hold before production use.

Cue

Mark beats and actions

Add beat, dialogue, action, camera, sound, or note markers. You can generate a beat-marker track from a BPM value you enter. CelCue does not import, analyse, play, or synchronise audio or video.

Rehearse

Inspect the timing plan

Scrub or run the visual playhead at ½×, 1×, or 2× to inspect active drawings and cues. Rehearsal visualises the entered plan; it is not rendered animation, production playback, lip-sync, or delivery certification.

Check

Review diagnostics

CelCue flags structural gaps, overlaps, out-of-range blocks, and long holds in the data you entered. These checks are review prompts, not artistic judgements or automatic corrections.

Handoff

Export or back up deliberately

Share a summary or export CSV, readable timing text, PDF, or a CelCue JSON backup through Apple's system interfaces. JSON restore reads only a file you choose. Keep an independent backup and verify every exported or restored file before relying on it.

Contact

Still need help?

Email compactrefinedapps@outlook.com and include your device model, iOS or iPadOS version, CelCue version, and a short description. Do not attach confidential scripts, production material, passwords, payment information, or other sensitive information.
