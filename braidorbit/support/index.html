Keep your place.

BraidOrbit is an offline iPhone and iPad companion for planning and resuming the common round eight-cord Kumihimo disk braid. No account, subscription, advertising, or network connection is required.

Set up

Match the physical disk

Create a project, choose a colour for each numbered cord, and place the cords on your physical disk using the same TOP reference shown in BraidOrbit.

Estimate

Calibrate before cutting

Enter the finished length, multiplier, and finishing allowance. The cut list is a planning estimate: cord, beads, tension, structure, and finishing method change real consumption. Make a short sample before cutting valuable material.

Resume

Return to the next move

Open the saved project and choose Braid. Tap Done only after completing the displayed move. When pausing, leave the cords seated in the disk; use Undo if the last recorded move was not completed.

Scope

Which braid is supported?

Version 1.0 covers the common round eight-cord handheld-disk rhythm. It is not a bead-pattern designer, finished-pattern preview, marudai guide, or universal Kumihimo generator.
